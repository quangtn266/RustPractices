# Introdution.
1. Generate a secret number (random number library 1 -> 100).
2. Type an input number (guess number).
   1. if guess > secret_num ===> print "Too big".
   2. if guess < secret_num ===> "Too small"
   3. if guess == secret_num ===> "You win"

# Configuration.

```
cargo run
```
